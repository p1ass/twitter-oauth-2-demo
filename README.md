# twitter-oauth-2-demo

Twitter API v2で追加されたOAuth2.0 Authorization Code Flowを使った認可のデモ用アプリケーション

## Blog

https://blog.p1ass.com/posts/twitter-api-oauth-2/

## Getting Started

1. 環境変数を設定する

- `TWITTER_CLIENT_ID`
- `TWITTER_CLIENT_SECRET`

2. 起動する

```shell
$ go run main.go
```

3. 表示されるリンクをクリックする
